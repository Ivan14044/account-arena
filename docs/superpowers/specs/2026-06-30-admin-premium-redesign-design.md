# Admin Premium Redesign — Design Spec («Aurora Light»)

## Goal
Полный редизайн админ-панели `/admin` (Laravel Blade + AdminLTE 3) до
**премиального, единого, выровненного по сетке и адаптивного** вида. Текущий вид —
«разбой» (несогласованные per-page стили, светло-серая гамма с синим #4e73df,
которая не нравится). Нужен единый «почерк» на всех ~74 страницах.

## Direction (выбрано пользователем)
**Светлая премиум** — эстетика Linear/Stripe: тёплый off-white фон, белые карточки
с мягкими тенями и скруглением, индиго-акцент, много воздуха, чистая сетка.

## Design tokens (`public/assets/admin/css/premium.css`, `:root`)
- **Поверхности:** bg `#f5f6fb` (+ радиальный indigo-glow), surface `#fff`,
  surface-2 `#f8f9fc`, sidebar `#fff`.
- **Текст:** `#1e2433` / muted `#64708a` / subtle `#97a1b5`.
- **Границы:** `#e8ebf3` / strong `#dde1ec`.
- **Акцент:** индиго `#4f46e5` (600/700), soft `#eef2ff`, градиент `#6366f1→#8b5cf6`.
- **Семантика:** success/warning/danger/info — насыщенный + soft-фон бейджей.
- **Радиусы:** 8/10/12/16/20px + pill. **Тени:** многослойные мягкие. **Шрифт:** Inter.

## Architecture
- Одна авторитетная таблица стилей `premium.css` — единый источник правды.
- Грузится **глобально** на каждой странице через AdminLTE-плагин `AAPremium`
  (`config/adminlte.php`, `active => true`), последним — чтобы перекрывать
  ядро AdminLTE и другие плагины.
- Переопределяет все компоненты: сайдбар (светлый, активный = индиго-soft +
  левая планка), топбар, карточки, кнопки (primary = индиго-градиент), формы
  (фокус-ринг индиго), таблицы (тонкая uppercase-шапка, hover, бейджи), stat-карточки,
  хедеры, пагинацию, модалки, alert'ы, dropdown'ы, empty-state.
- Mobile-first: брейкпоинты 991/575px (сайдбар-оффканвас, карточки в стек,
  таблицы скроллом, кнопки на всю ширину).

## Rollout
1. **Фундамент** — `premium.css` + глобальная регистрация. Трансформирует все
   страницы разом. ✅
2. **Зачистка** — нейтрализовать старый `layouts/modern-styles.blade.php` и
   конфликтующие per-page `<style>`, чтобы остался единственный источник стилей.
3. **Полировка по странице** — выровнять разметку по сетке/компонентам, где нужно.
4. **Верификация** — каждая страница на десктопе + 375px (true emulation).

## Out of scope
- Логика/маршруты/контроллеры/данные — не трогаем (behaviour-preserving).
- Публичный фронтенд (SPA) — отдельная история (уже сделан в `feat/web-*`).
- Кабинет поставщика — те же adminlte-стили унаследует косметически; проверить,
  что не сломалось, прицельно не редизайнить (вне scope запроса про `/admin`).
